# calculator
This is a calculator that performs the 4 basic math functions: addition, subtraction, division, and multiplication. This project is called "MyMelody Calculator". I based the theme on the Sanrio franchise. I thought the shades of pink were really pretty!

I created this calculator since I couldn't take AP Computer Science this year at my school due to the way I have to get my credits, and this is one of the projects that is made in that class. I will say that it was challenging to make since I had to use onclick attributes, vh elements, and JavaScript for the first time. 
I really struggled with the JavaScript portion of this project since I didn't know how to make it work as a function in the website; I'm used to using the anchor elements to connect HTML pages, so this was an entirely new beast for me. To overcome this struggle, I followed a tutorial provided by Bro Code on YouTube! He is really in-depth with his tutorials, explaining each step of his code.

I made this project by using HTML, CSS, and JavaScript. HTML was used for the button numbers, CSS styled the visuals of the calculator, and JavaScript was used to provide functions to the calculator.
